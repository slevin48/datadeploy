datadeploy

Short description

Syntax

datadeploy(header,data,herokuapp)

Arguments

header

header of the variable to deploy (vector of strings)

data

core of the data to deploy (matrix of string or numbers)

herokuapp

name of heroku app to deploy (contained in the URL)

Description

Deploy your data as an API with a JSON format on Heroku.

Examples

M = ['A','B';
     '1','2';
     '3','4']

header = M(1,:)
data = M(2:$,:)
herokuapp = "myherokuapp"
datadeploy(header,data,herokuapp);
